# PROYECTO NOC

# dev 

1. Renombrar el archivo env.example a .env
2. Configurar las variables de entorno
3. Ejecutar el comando ```npm install```
4. Configurar y levantar las bases de datos con el comando
   ```
   docker-compose up -d
   ```
5. Ejecutar ```npm run dev```
